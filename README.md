# CST-DDOS-tool-python
# 📋 项目概述
## 数据包发送器是一个基于Python开发的网络压力测试工具，主要用于向指定目标发送大量UDP数据包。该工具支持多线程并发发送，可自定义发送速度和线程数量，适用于网络设备性能测试和网络通信调试等场景。
# 🚀 主要功能

###    ‌多线程支持‌：可配置1-64个并发线程同时发送数据包
###     ‌速度控制‌：支持1-2048级发送速度调节
###     ‌实时监控‌：显示每个线程的数据包发送统计

# ⚙️ 技术特点

###     使用Python原生socket库实现网络通信
###     实时状态显示便于监控发送进度
###     轻量级设计，无需复杂依赖环境

# 📋 使用方法

###     运行程序后输入目标IP地址/域名
###     设置目标端口号
###     配置发送速度(1-2048级)
###     设置线程数量(1-64个)
###     程序开始多线程发送数据包

# ⚠️ 使用声明

###     本工具仅限用于合法的网络测试场景
###     请勿用于任何非法网络攻击活动
###     使用前请确保获得目标网络的测试授权
###     开发者不对滥用行为承担任何责任

# 🔧 系统要求

###     Python 3.0及以上版本
###     支持Windows、Linux、macOS等操作系统
###     无需额外第三方依赖库

# 📊 工具优势

###     ‌高效并发‌：多线程设计提高发送效率
###     ‌灵活配置‌：速度和线程数可自由调节
###     ‌实时反馈‌：每个线程独立显示发送状态
###     ‌易于使用‌：简洁的交互式命令行界面
###     ‌跨平台‌：支持主流操作系统运行
